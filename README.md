# Quicky AI

Quicky AI is a modern AI-powered content creation web application frontend built using React.js and Tailwind CSS.

##  Project Overview

This project is a frontend implementation of an AI content creation platform where users can:
- Write articles using AI UI flow
- Generate content (UI only)
- Sign In / Sign Up
- Continue with Google authentication (UI)
- Navigate through dashboard-style interface

##  Features

- Modern Landing Page UI
- Responsive Design (Mobile + Desktop)
- Authentication Pages (Login / Signup)
- Google Sign-In UI
- Dashboard UI structure
- Clean and modern UX design
- Smooth navigation flow (frontend only)

##  Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- React Router DOM

##  Project Status

Backend is currently under development.  
This repository contains only the frontend implementation.
